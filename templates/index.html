📖 nmap Reference and Scanning Guide

Essential Commands

nmap -sV -oX output.xml 192.168.1.0/24

Version detection. The recommended starting point. enables vulnerability matching in NmapViz.

nmap -A -oX output.xml 192.168.1.0/24

Aggressive: OS detection + version + default scripts + traceroute. More data, more noise.

nmap -sV --script vuln -oX output.xml [target]

Vulnerability scripts: EternalBlue, Heartbleed, SMB issues, FTP anon, RDP checks, etc.

nmap -sV --script default,vuln -oX output.xml [target]

Default + vuln scripts combined. Best coverage/time balance.

Performance. Large Networks

FlagEffect
--top-ports 100Only scan the 100 most common ports. Up to 10x faster on large subnets.
-T4Aggressive timing. Faster with minor accuracy tradeoff. Ideal for stable LANs.
-T5Insane timing. Only on localhost or very stable networks.
--min-rate 1000Send at least 1000 pkts/s. Good for fast internal scans.
-nSkip DNS resolution. Significant speed boost.
--openShow only open ports. Reduces output considerably.
-PnSkip host discovery. Use when ICMP is blocked.
--exclude [IPs]Exclude specific hosts (printers, fragile devices, etc).

Large Network Strategy

nmap -sV --top-ports 150 -T4 -n --open -oX fast.xml 10.0.0.0/16

Fast wide-coverage scan for /16 or larger. Top 150 ports, aggressive timing, no DNS.

nmap -sV --top-ports 500 -T4 -oX scan.xml 192.168.0.0/24

Thorough /24 scan. Good balance between coverage and time (5-15 min).

nmap -sV -p- -T4 -oX full.xml [single-target]

Full port scan (all 65535). Only feasible on individual hosts.

Multi-File Scanning (NmapViz feature)

nmap -sV -oX scan_192.xml 192.168.1.0/24 nmap -sV -oX scan_10.xml 10.0.0.0/24

Scan different subnets separately, then drop both XMLs into NmapViz at once. They are merged automatically.

Useful NSE Scripts

Script / CategoryWhat it detects
--script vulnKnown CVEs: EternalBlue, Heartbleed, SMB issues, Shellshock, and more.
--script smb-security-modeSMB signing status. Critical for NTLM relay detection.
--script ftp-anonFTP anonymous login check.
--script ssh-auth-methodsSSH authentication methods (password vs key).
--script rdp-enum-encryptionRDP NLA and encryption checks.
--script dns-zone-transferDNS AXFR attempt.
--script http-default-accountsDefault credentials on web interfaces.
--script ipmi-version,ipmi-bruteIPMI/BMC detection and auth bypass.
--script redis-infoRedis unauthenticated access check.

Stealth and Evasion

FlagEffect
-sSSYN scan (default). Half-open, less likely to be logged.
-T1 / -T2Paranoid/sneaky timing. Very slow but harder to detect.
--randomize-hostsScan hosts in random order.
-D RND:5Decoy scan with 5 random source IPs.
-fFragment packets. Bypasses some firewalls.

Only use evasion techniques on networks you are authorised to test.
